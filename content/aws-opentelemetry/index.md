Recently AWS announced the end of support for X-Ray SDK and daemon. So they are recommending

> Before February 25th, 2027 we recommend that you migrate to OpenTelemetry solutions for instrumenting your application and sending traces to AWS X-Ray
source: [AWS X-Ray SDK and daemon end of support](https://docs.aws.amazon.com/xray/latest/devguide/xray-daemon-eos.html)

AWS OpenTelemetry is a open source project that provides a set of tools to instrument, collect, and export telemetry data. It is a way to collect telemetry data from your applications and services and send it to a telemetry backend.

* Sending OpenTelemetry traces, metrics and logs from your Lambda and ECS workloads to AWS X-Ray
